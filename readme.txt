# Django Project README

## Project Overview

This Django rest api project demonstrate how to register and login user using JWT TOKEN.


## Prerequisites

Before you begin, ensure you have met the following requirements:

- Python (3.12.0 recommended)
- Django (install via `pip install Django`)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/njoro-njoroge/django-rest-api-jwt

2. Navigate to the project directory:
   cd django-rest-api-jwt

3. Create a virtual environment (optional but recommended):
   python -m venv venv

4.  Activate virtual environment
    On Windows run venv\Scripts\activate
    On MacOS  source venv/bin/activate

5. Install project dependencies
    pip install -r requirements.txt

6. Apply database migrations:
   python manage.py makemigrations
   python manage.py migrate

5. Run development server
   python manage.py runserver

   The application should be accessible at http://127.0.0.1:8000/ in your web browser.
